# 🔴 Red Ball - 2D Platform Oyunu

**Can ÇETİN anısına geliştirilmiştir.**

Red Ball 4'ten ilham alınarak HTML5 Canvas ve JavaScript ile geliştirilmiş 2D platform oyunu.

## 🎮 Oyna

Oyunu doğrudan tarayıcıda oynayabilirsiniz!

## 🕹️ Kontroller

| Tuş | Aksiyon |
|-----|---------|
| ← → Ok Tuşları / A-D | Sağa-sola hareket |
| Space / Yukarı Ok / W | Zıplama |
| Shift | Hızlı koşma (1.8x) |
| ESC | Duraklatma |

## 🏔️ Bölümler

1. **Yeşil Vadi** - Başlangıç bölümü
2. **Karanlık Orman** - Orta zorluk
3. **Lav Dağı** - Zor bölüm

## ✨ Özellikler

- 🔴 Sevimli kırmızı top karakteri
- 👾 Mor kare düşmanlar
- ⭐ Yıldız toplama sistemi
- 🏁 Bölüm sonu bayrağı
- 💀 Dikenler ve engeller
- 🎆 Parçacık efektleri
- 📱 Mobil uyumluluk
- 🏃 Sprint (hızlanma) mekaniği

## 🚀 Kurulum

```bash
pip install -r requirements.txt
streamlit run app.py
```
